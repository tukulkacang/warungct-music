🍗 Warung CT - Aplikasi Pemesanan Online

Aplikasi web sederhana untuk memudahkan pelanggan Warung CT (Tikung, Lamongan) melakukan pemesanan via WhatsApp dengan fitur kalkulasi ongkir otomatis berbasis lokasi (GPS).

🚀 Fitur Utama

GPS Distance Tracking: Menghitung jarak otomatis dari lokasi pelanggan ke Warung CT.

Smart Shipping Cost: Skema ongkir otomatis (KM pertama Rp 2.000, selanjutnya Rp 1.000/km).

Interactive Menu: Pilihan menu utama dan tambahan yang mudah digunakan.

WhatsApp Integration: Pesanan otomatis terformat dan dikirim langsung ke admin.

QR Code Share: Fitur berbagi link aplikasi melalui QR Code yang mudah di-scan.
